# Roadmap

## フェーズ0: MVP（現在）

- 起動時 `status` による認証チェック
- 未認証時のログイン導線
- `list` による一覧表示
- サムネイル + ホバーGIF + 選択でHLS再生

## フェーズ1: 基本操作の拡張

- 右クリックメニューから削除（`delete <asset_id> [--force]`）
- 動画詳細の表示（`show <asset_id>`）
- “URLコピー”
  - HLS URL
  - サムネイルURL
  - （可能なら）MP4 URL

## フェーズ2: アップロード

- `upload <file> --progress` の進捗表示
- 失敗時の再試行/部分的復帰

## フェーズ3: 品質/運用

- ログ保存（ユーザー報告を容易に）
- 設定画面（プロキシ・タイムアウト等）※必要になったら

## 優先順位の判断基準

- CLI契約が安定しているか（非対話ログイン/機械可読進捗など）
- UI上のエラーが復帰可能か（ユーザーが自力で解決できる導線があるか）
- 追加機能が既存の責務境界（Main/Preload/Renderer）を壊さないか
