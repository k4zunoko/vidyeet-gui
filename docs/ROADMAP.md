# Roadmap

## フェーズ0: MVP（現実装に含まれる）

- 起動時 `status` による認証チェック
- 未認証時のログイン導線
- `list` による一覧表示
- サムネイル + ホバーGIF + 選択でHLS再生

フェーズ0に含めた拡張（現実装）:

- ログアウト
- 右クリックメニュー
  - MP4リンクコピー
  - 削除（確認ダイアログあり）
- アップロード（ファイル選択 + 進捗表示 + 完了後リロード）

## フェーズ1: 基本操作の拡張

- 動画詳細の表示（`show <asset_id>`）
- “URLコピー”の拡張
  - HLS URL
  - サムネイルURL
  - MP4 URL（現状は対応済み）

## フェーズ2: アップロード

- 失敗時の再試行/部分的復帰
- キャンセル/タイムアウト
- 進捗（percent）の表示強化

## フェーズ3: 品質/運用

- ログ保存（ユーザー報告を容易に）
- 設定画面（プロキシ・タイムアウト等）※必要になったら

## 優先順位の判断基準

- CLI契約が安定しているか（非対話ログイン/機械可読進捗など）
- UI上のエラーが復帰可能か（ユーザーが自力で解決できる導線があるか）
- 追加機能が既存の責務境界（Main/Preload/Renderer）を壊さないか
