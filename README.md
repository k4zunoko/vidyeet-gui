# Vidyeet - 容量制限を気にしない動画共有

DiscordやX(Twitter)など、容量制限のあるSNSで動画を共有する際の「ファイルが大きすぎて送信できません」を解決します。

**Vidyeet**は、動画をクラウドにアップロードしてリンクを共有するシンプルなWindowsアプリです。

---

## 📥 ダウンロード・インストール

### 1. インストーラーをダウンロード

1. **[「Releases」](../../releases)** をクリック
2. 最新のバージョンを選択
3. `Vidyeet-x.x.x-Setup.exe` をダウンロード

### 2. アプリをインストール

ダウンロードした `.exe` ファイルを実行するだけです。管理者権限は不要です。

---

## 🚀 使い方

### 初回セットアップ（必要な情報）

初回起動時にMuxアカウントの認証情報を入力します：

| 必要な情報 | 入手方法 |
|-----------|---------|
| **Token ID** | [Mux Dashboard](https://dashboard.mux.com/) → Settings → Access Tokens |
| **Token Secret** | 同上（Secretは初回表示時のみ確認可能） |

> 💡 **Muxアカウントがない場合**：[mux.com](https://mux.com) で無料登録できます。

---

### 動画をアップロードして共有

#### 方法1: アップロードボタンから
1. アプリ上部の **「+ アップロード」** ボタンをクリック
2. 動画ファイルを選択
3. アップロード完了を待つ
4. 動画を右クリック → **「リンクをコピー」** で共有URLを取得

#### 方法2: ドラッグ&ドロップ
1. エクスプローラーで動画ファイルを選択
2. VidYeetウィンドウにドラッグ&ドロップ
3. 自動でアップロードが開始されます

> 📝 **複数ファイルもOK**：複数の動画をまとめてドラッグ&ドロップすると、順番にアップロードされます。

---

### 動画を見る

アップロード済みの動画はサムネイルで一覧表示されます：

| 操作 | 方法 |
|------|------|
| **プレビュー** | サムネイルにマウスを乗せるとGIFで自動プレビュー |
| **再生** | サムネイルをクリックすると動画プレイヤーで再生 |
| **詳細情報** | 再生中に表示されるパネルで時間・解像度・作成日を確認 |

---

### 動画を削除

右クリックメニューから **「削除」** を選択します。

⚠️ **注意**：削除は元に戻せません。ご注意ください。

---

## 💡 Vidyeetで解決できること

| お悩み | Vidyeetの解決策 |
|--------|----------------|
| Discordで「25MBを超えています」エラー | 制限を気にせず動画を共有 |
| Twitterで動画が画質低下する | オリジナル品質のままリンク共有 |
| LINEで長い動画が送れない | リンク形式なら長さ関係なし |
| 複数のSNSに同じ動画を投稿したい | 1つのリンクをどこにでも貼れる |

## 🖥️ 動作環境

- **OS**: Windows 10/11（64bit）
- **必要な環境**: インターネット接続、Muxアカウント

- このアプリは **Mux** の動画ホスティングサービスを利用しています
- アップロードされた動画はMuxのサーバーに保存されます
